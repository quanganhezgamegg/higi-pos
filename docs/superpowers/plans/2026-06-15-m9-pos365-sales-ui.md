## Thiết kế chi tiết

### Quick-add logic
- Product có **đúng 1 size** (`sizes.length === 1`) VÀ **không có topping** → tap card = thêm ngay vào cart với `{size: sizes[0], sugar: "70%", ice: "Vừa", qty: 1}` mà không mở dialog.
- Product có **nhiều size** HOẶC **có topping** → tap card = mở `OptionDialog` (giữ nguyên từ M3).
- Cart đã có item cùng `product_id + size_name` với sugar/ice/topping mặc định → tăng `quantity` thay vì tạo dòng mới (stacking).

### Tìm kiếm sản phẩm
- Client-side filter trên `products[]` bằng `query.toLowerCase()` khớp `product.name`.
- Khi có query: ẩn filter danh mục, hiển thị kết quả xuyên tất cả danh mục.
- Debounce 150 ms để tránh re-render liên tục khi gõ.

### Order type + bàn selector (TopBar)
- Dropdown loại đơn: `TAKEAWAY` = "Mang đi", `DINE_IN` = "Tại bàn".
- Khi `DINE_IN`: hiện dropdown bàn trống kề bên, placeholder "Chọn bàn".
- Hiển thị tên bàn đang chọn trong top bar: `[Tại bàn ▼] [Bàn 5 ▼]`.

### Cart panel
- Mỗi line item: tên + mô tả (size/sugar/ice/topping), stepper `− qty +` inline, giá dòng.
- Long-press hoặc nút `×` để xóa dòng.
- Footer cố định: tổng + nút "THANH TOÁN" (`h-14`, full-width, màu primary).
- Đơn đang mở (open orders) hiện trong `ScrollArea` phía trên footer — nhỏ gọn hơn hiện tại.

```typescript
import { create } from "zustand"
import type { OrderItemInput, OrderItemToppingInput } from "@/lib/api/orders"

export type CartItem = OrderItemInput & { key: string }

/** Tính tiền một dòng: (unit_price + tổng topping) × qty − line_discount */
export function calcLine(item: OrderItemInput): number {
  const toppingTotal = item.toppings.reduce(
    (sum, t) => sum + t.price * t.quantity,
    0,
  )
  return Math.max(0, (item.unit_price + toppingTotal) * item.quantity - item.line_discount)
}

/** Kiểm tra hai item có "cùng cấu hình" để stack không */
function isSameConfig(a: OrderItemInput, b: OrderItemInput): boolean {
  if (a.product_id !== b.product_id) return false
  if (a.size_name !== b.size_name) return false
  if (a.sugar_level !== b.sugar_level) return false
  if (a.ice_level !== b.ice_level) return false
  if (a.line_note !== b.line_note) return false
  if (a.toppings.length !== b.toppings.length) return false
  const sortedA = [...a.toppings].sort((x, y) => (x.topping_id ?? 0) - (y.topping_id ?? 0))
  const sortedB = [...b.toppings].sort((x, y) => (x.topping_id ?? 0) - (y.topping_id ?? 0))
  return sortedA.every((t, i) => t.topping_id === sortedB[i].topping_id)
}

type CartState = {
  items: CartItem[]
  /** Thêm item; nếu cùng cấu hình với dòng đã có → tăng qty (stacking) */
  addItem: (item: OrderItemInput) => void
  removeItem: (key: string) => void
  setQty: (key: string, qty: number) => void
  clear: () => void
}

export const useCartStore = create<CartState>((set) => ({
  items: [],

  addItem: (item) =>
    set((state) => {
      const existing = state.items.find((i) => isSameConfig(i, item))
      if (existing) {
        return {
          items: state.items.map((i) =>
            i.key === existing.key ? { ...i, quantity: i.quantity + item.quantity } : i,
          ),
        }
      }
      return { items: [...state.items, { ...item, key: crypto.randomUUID() }] }
    }),

  removeItem: (key) =>
    set((state) => ({ items: state.items.filter((i) => i.key !== key) })),

  setQty: (key, qty) =>
    set((state) => ({
      items: state.items.map((i) =>
        i.key === key ? { ...i, quantity: Math.max(1, qty) } : i,
      ),
    })),

  clear: () => set({ items: [] }),
}))
```

```typescript
import { beforeEach, describe, expect, it } from "vitest"
import { act, renderHook } from "@testing-library/react"
import { useCartStore, calcLine } from "@/store/cartStore"
import type { OrderItemInput } from "@/lib/api/orders"

function makeItem(overrides: Partial<OrderItemInput> = {}): OrderItemInput {
  return {
    product_id: 1,
    product_name: "Cà phê sữa",
    size_name: "M",
    unit_price: 45000,
    quantity: 1,
    sugar_level: "70%",
    ice_level: "Vừa",
    line_note: null,
    line_discount: 0,
    toppings: [],
    ...overrides,
  }
}

beforeEach(() => {
  // reset store giữa các test
  useCartStore.setState({ items: [] })
})

describe("calcLine", () => {
  it("trả về unit_price × qty khi không có topping và không discount", () => {
    expect(calcLine(makeItem({ quantity: 2, unit_price: 45000 }))).toBe(90000)
  })

  it("cộng topping vào giá dòng", () => {
    const item = makeItem({
      unit_price: 45000,
      quantity: 1,
      toppings: [{ topping_id: 1, topping_name: "Trân châu", price: 5000, quantity: 1 }],
    })
    expect(calcLine(item)).toBe(50000)
  })

  it("trừ line_discount", () => {
    expect(calcLine(makeItem({ unit_price: 45000, quantity: 1, line_discount: 5000 }))).toBe(40000)
  })

  it("không bao giờ trả về số âm", () => {
    expect(calcLine(makeItem({ unit_price: 10000, quantity: 1, line_discount: 99999 }))).toBe(0)
  })
})

describe("useCartStore — addItem", () => {
  it("thêm item mới vào items", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem()))
    expect(result.current.items).toHaveLength(1)
    expect(result.current.items[0].product_name).toBe("Cà phê sữa")
  })

  it("stack item cùng cấu hình — tăng qty thay vì tạo dòng mới", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem({ quantity: 1 })))
    act(() => result.current.addItem(makeItem({ quantity: 2 })))
    expect(result.current.items).toHaveLength(1)
    expect(result.current.items[0].quantity).toBe(3)
  })

  it("KHÔNG stack item khác size", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem({ size_name: "M" })))
    act(() => result.current.addItem(makeItem({ size_name: "L" })))
    expect(result.current.items).toHaveLength(2)
  })
})

describe("useCartStore — removeItem / setQty / clear", () => {
  it("removeItem xóa đúng dòng", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem()))
    const key = result.current.items[0].key
    act(() => result.current.removeItem(key))
    expect(result.current.items).toHaveLength(0)
  })

  it("setQty cập nhật số lượng", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem()))
    const key = result.current.items[0].key
    act(() => result.current.setQty(key, 5))
    expect(result.current.items[0].quantity).toBe(5)
  })

  it("setQty không cho qty xuống dưới 1", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem()))
    const key = result.current.items[0].key
    act(() => result.current.setQty(key, 0))
    expect(result.current.items[0].quantity).toBe(1)
  })

  it("clear xóa tất cả items", () => {
    const { result } = renderHook(() => useCartStore())
    act(() => result.current.addItem(makeItem()))
    act(() => result.current.addItem(makeItem({ product_id: 2, size_name: "L" })))
    act(() => result.current.clear())
    expect(result.current.items).toHaveLength(0)
  })
})
```

```tsx
import { useState } from "react"
import { Button } from "@/components/ui/button"
import {
  Dialog,
  DialogContent,
  DialogFooter,
  DialogHeader,
  DialogTitle,
} from "@/components/ui/dialog"
import { Input } from "@/components/ui/input"
import { DEFAULT_ICE_LEVELS, DEFAULT_SUGAR_LEVELS } from "@/lib/constants"
import { formatVnd } from "@/lib/format"
import { calcLine } from "@/store/cartStore"
import type { OrderItemInput } from "@/lib/api/orders"
import type { Product, ProductSize, Topping } from "@/lib/api/menu"

type Props = {
  product: Product
  toppings: Topping[]
  onClose: () => void
  onAdd: (item: OrderItemInput) => void
}

export function OptionDialog({ product, toppings, onClose, onAdd }: Props) {
  const defaultSize = product.sizes.find((s) => s.is_default) ?? product.sizes[0]
  const [size, setSize] = useState<ProductSize | undefined>(defaultSize)
  const [sugar, setSugar] = useState(DEFAULT_SUGAR_LEVELS[3]) // "70%"
  const [ice, setIce] = useState(DEFAULT_ICE_LEVELS[2])       // "Vừa"
  const [qty, setQty] = useState(1)
  const [selectedToppings, setSelectedToppings] = useState<Topping[]>([])
  const [note, setNote] = useState("")

  const unitPrice = product.base_price + (size?.price_delta ?? 0)
  const item: OrderItemInput = {
    product_id: product.id,
    product_name: product.name,
    size_name: size?.name ?? null,
    unit_price: unitPrice,
    quantity: qty,
    sugar_level: sugar,
    ice_level: ice,
    line_note: note.trim() || null,
    line_discount: 0,
    toppings: selectedToppings.map((t) => ({
      topping_id: t.id,
      topping_name: t.name,
      price: t.price,
      quantity: 1,
    })),
  }

  return (
    <Dialog open onOpenChange={(open) => !open && onClose()}>
      <DialogContent className="max-w-xl">
        <DialogHeader>
          <DialogTitle>{product.name}</DialogTitle>
        </DialogHeader>
        <div className="flex flex-col gap-4">
          <OptionGroup label="Size">
            {product.sizes.map((s) => (
              <Button
                key={s.id}
                variant={size?.id === s.id ? "default" : "outline"}
                className="h-11 min-w-[64px]"
                onClick={() => setSize(s)}
              >
                {s.name}
                {s.price_delta > 0 ? ` +${formatVnd(s.price_delta)}` : ""}
              </Button>
            ))}
          </OptionGroup>
          <OptionGroup label="Đường">
            {DEFAULT_SUGAR_LEVELS.map((level) => (
              <Button
                key={level}
                variant={sugar === level ? "default" : "outline"}
                className="h-11"
                onClick={() => setSugar(level)}
              >
                {level}
              </Button>
            ))}
          </OptionGroup>
          <OptionGroup label="Đá">
            {DEFAULT_ICE_LEVELS.map((level) => (
              <Button
                key={level}
                variant={ice === level ? "default" : "outline"}
                className="h-11"
                onClick={() => setIce(level)}
              >
                {level}
              </Button>
            ))}
          </OptionGroup>
          <OptionGroup label="Topping">
            {toppings.map((t) => {
              const active = selectedToppings.some((x) => x.id === t.id)
              return (
                <Button
                  key={t.id}
                  variant={active ? "default" : "outline"}
                  className="h-11"
                  onClick={() =>
                    setSelectedToppings((prev) =>
                      active ? prev.filter((x) => x.id !== t.id) : [...prev, t],
                    )
                  }
                >
                  {t.name} +{formatVnd(t.price)}
                </Button>
              )
            })}
          </OptionGroup>
          <div>
            <p className="mb-2 text-sm font-medium">Ghi chú</p>
            <Input
              value={note}
              onChange={(e) => setNote(e.target.value)}
              placeholder="Ít ngọt, không đá..."
            />
          </div>
          <div className="flex items-center justify-between rounded-lg border p-3">
            <div className="flex items-center gap-3">
              <Button
                variant="outline"
                size="icon"
                className="size-11"
                onClick={() => setQty((v) => Math.max(1, v - 1))}
              >
                −
              </Button>
              <span className="w-10 text-center text-lg font-bold">{qty}</span>
              <Button
                variant="outline"
                size="icon"
                className="size-11"
                onClick={() => setQty((v) => v + 1)}
              >
                +
              </Button>
            </div>
            <span className="text-lg font-bold">{formatVnd(calcLine(item))}</span>
          </div>
        </div>
        <DialogFooter>
          <Button variant="outline" onClick={onClose}>
            Hủy
          </Button>
          <Button className="h-11 px-6" onClick={() => onAdd(item)}>
            Thêm vào giỏ
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}

function OptionGroup({ label, children }: { label: string; children: React.ReactNode }) {
  return (
    <div>
      <p className="mb-2 text-sm font-medium">{label}</p>
      <div className="flex flex-wrap gap-2">{children}</div>
    </div>
  )
}
```

```tsx
function canQuickAdd(product: Product, allToppings: Topping[]): boolean {
  return product.sizes.length === 1 && allToppings.length === 0
}

export function ProductGrid({ products, toppings, onOpenDialog }: Props) {
  const addItem = useCartStore((s) => s.addItem)

    if (canQuickAdd(product, toppings)) {
      const size = product.sizes[0]
      const item: OrderItemInput = {
        product_id: product.id,
        product_name: product.name,
        size_name: size.name,

        quantity: 1,
        sugar_level: DEFAULT_SUGAR_LEVELS[3], // "70%"
        ice_level: DEFAULT_ICE_LEVELS[2],     // "Vừa"
        line_note: null,
        line_discount: 0,
        toppings: [],
      }
      addItem(item)
    } else {
      onOpenDialog(product)
    }
  }

    return (
      <div className="flex flex-1 items-center justify-center text-sm text-muted-foreground">

          toppings={toppings}
```

```tsx

import { useSearchParams } from "react-router-dom"
import { DEFAULT_SUGAR_LEVELS, DEFAULT_ICE_LEVELS } from "@/lib/constants"
import {
  listCategories,
  listProducts,
  listToppings,
  type Category,
  type Product,
  type Topping,
} from "@/lib/api/menu"
import { listOpenOrders, type Order, type OrderType } from "@/lib/api/orders"
import { getCurrentShift, type Shift } from "@/lib/api/shifts"
import { listTableStatus, type TableStatus } from "@/lib/api/tables"
import { useCartStore } from "@/store/cartStore"
import { TopBar } from "@/routes/sales/TopBar"
import { CategoryPane } from "@/routes/sales/CategoryPane"
import { ProductGrid } from "@/routes/sales/ProductGrid"
import { CartPanel } from "@/routes/sales/CartPanel"
import { OptionDialog } from "@/routes/sales/OptionDialog"
import { Button } from "@/components/ui/button"
import { Link } from "react-router-dom"

export default function Sales() {
  const [searchParams] = useSearchParams()
  const requestedTableId = Number(searchParams.get("tableId")) || null

  // --- Data state ---
  const [shift, setShift] = useState<Shift | null | undefined>(undefined)
  const [categories, setCategories] = useState<Category[]>([])
  const [products, setProducts] = useState<Product[]>([])
  const [toppings, setToppings] = useState<Topping[]>([])
  const [tables, setTables] = useState<TableStatus[]>([])
  const [openOrders, setOpenOrders] = useState<Order[]>([])

  // --- UI state ---
  const [categoryId, setCategoryId] = useState<number | null>(null)
  const [orderType, setOrderType] = useState<OrderType>(
    requestedTableId ? "DINE_IN" : "TAKEAWAY",
  )
  const [tableId, setTableId] = useState<number | null>(requestedTableId)
  const [searchQuery, setSearchQuery] = useState("")
  const [dialogProduct, setDialogProduct] = useState<Product | null>(null)
  const [error, setError] = useState<string | null>(null)
  const [loading, setLoading] = useState(false)

  const addItem = useCartStore((s) => s.addItem)

  async function loadAll() {
    setError(null)
    try {
      const [currentShift, cats, prods, tops, tbls, orders] = await Promise.all([
        getCurrentShift(),
        listCategories(false),
        listProducts(null, false),
        listToppings(false),
        listTableStatus(),
        listOpenOrders(),
      ])
      setShift(currentShift)
      setCategories(cats)
      setProducts(prods)
      setToppings(tops)
      setTables(tbls)
      setOpenOrders(orders)
      setCategoryId((cur) => cur ?? cats[0]?.id ?? null)
    } catch (e) {
      setError(String(e))
      setShift(null)
    }
  }

  useEffect(() => {
    void loadAll()
  }, [])

  // Debounced search: 150ms
  const [debouncedQuery, setDebouncedQuery] = useState("")
  useEffect(() => {
    const t = setTimeout(() => setDebouncedQuery(searchQuery), 150)
    return () => clearTimeout(t)
  }, [searchQuery])

  const visibleProducts = useMemo(() => {
    if (debouncedQuery.trim()) {
      const q = debouncedQuery.toLowerCase()
      return products.filter((p) => p.name.toLowerCase().includes(q))
    }
    return products.filter((p) => categoryId === null || p.category_id === categoryId)
  }, [products, categoryId, debouncedQuery])

  const freeTables = tables.filter((s) => s.status === "TRONG")

  // --- Guard states ---
  if (shift === undefined) {
    return <div className="p-6 text-muted-foreground">Đang kiểm tra ca...</div>
  }

  if (shift === null) {
    return (
      <div className="flex min-h-screen flex-col items-center justify-center gap-4 p-6 text-center">
        <h1 className="text-2xl font-bold">Chưa mở ca làm việc</h1>
        <p className="max-w-md text-sm text-muted-foreground">
          Bạn cần mở ca trước khi bắt đầu bán hàng để đơn được gắn đúng ca đối soát.
        </p>
        <Button asChild className="h-12 px-6">
          <Link to="/shift">Mở ca ngay</Link>
        </Button>
        {error && <p className="text-sm text-destructive">{error}</p>}
      </div>
    )
  }

  return (
    <div className="flex h-screen flex-col bg-muted/20">
      {/* Top bar */}
      <TopBar
        orderType={orderType}
        tableId={tableId}
        freeTables={freeTables}
        searchQuery={searchQuery}
        shift={shift}
        onOrderTypeChange={setOrderType}
        onTableChange={setTableId}
        onSearchChange={setSearchQuery}
      />

      {/* 3-pane body */}
      <div className="flex min-h-0 flex-1 overflow-hidden">
        {/* Pane trái: danh mục */}
        <CategoryPane
          categories={categories}
          activeCategoryId={debouncedQuery ? null : categoryId}
          onSelect={(id) => {
            setCategoryId(id)
            setSearchQuery("")
          }}
        />

        {/* Pane giữa: lưới sản phẩm */}
        <main className="flex min-w-0 flex-1 flex-col overflow-hidden">
          <ProductGrid
            products={visibleProducts}
            toppings={toppings}
            onOpenDialog={setDialogProduct}
          />
        </main>

        {/* Pane phải: cart */}
        <CartPanel
          orderType={orderType}
          tableId={tableId}
          openOrders={openOrders}
          error={error}
          loading={loading}
          onError={setError}
          onLoadingChange={setLoading}
          onSubmitSuccess={() => void loadAll()}
        />
      </div>

      {/* Option dialog */}
      {dialogProduct && (
        <OptionDialog
          product={dialogProduct}
          toppings={toppings}
          onClose={() => setDialogProduct(null)}
          onAdd={(item) => {
            addItem(item)
            setDialogProduct(null)
          }}
        />
      )}
    </div>
  )
}
```

## Task 11: Xác minh thủ công (Manual Verification)

**Công cụ:** `npx tauri dev` (hoặc `pnpm tauri dev`)

- [ ] **Step 1: Khởi động ứng dụng** — `pnpm tauri dev`, chờ cửa sổ Tauri mở.
- [ ] **Step 2: Mở ca** — vào `/shift` mở ca nếu chưa có.
- [ ] **Step 3: Kiểm tra layout 3-pane** — truy cập `/sales`:
  - Cột trái hiện danh sách danh mục, highlight danh mục đang chọn.
  - Trung tâm hiện lưới card sản phẩm (tối thiểu 2 cột).
  - Cột phải là cart panel với tiêu đề "Đơn mang đi".
  - Top bar có toggle "Mang đi / Tại bàn" + ô tìm kiếm + badge ca.
- [ ] **Step 4: Quick-add** — bấm vào card sản phẩm có **1 size, không topping** → item xuất hiện ngay trong cart (không có dialog). Bấm lần 2 → qty tăng lên 2 (stacking).
- [ ] **Step 5: Dialog path** — bấm vào card sản phẩm có **nhiều size hoặc có topping** → `OptionDialog` mở. Chọn xong, nhấn "Thêm vào giỏ" → dialog đóng, item xuất hiện trong cart.
- [ ] **Step 6: Qty stepper** — trong cart, nhấn `+` / `−` trên dòng item → số lượng thay đổi, giá dòng cập nhật ngay.
- [ ] **Step 7: Xóa dòng** — nhấn nút `×` (Trash2) → dòng biến mất.
- [ ] **Step 8: Tìm kiếm** — gõ tên mon vào ô tìm kiếm → lưới lọc realtime xuyên danh mục. Xóa query → về lại lưới theo danh mục.
- [ ] **Step 9: Chọn bàn** — chuyển sang "Tại bàn", chọn bàn từ dropdown → header cart hiện "Đơn — Bàn X".
- [ ] **Step 10: Tạo đơn & thanh toán** — thêm ≥1 món, nhấn "THANH TOÁN" → navigate sang `/payment/:orderId`, không có lỗi.
- [ ] **Step 11: Touch targets** — tất cả nút có chiều cao/rộng ≥ 44px (kiểm tra bằng DevTools).
